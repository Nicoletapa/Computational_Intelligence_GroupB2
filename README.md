# Computational_Intelligence_GroupB2

This project was our final exam for the course Computational Intelligence fall 2025

## Quick Start

Run the visualization with:

```bash with MAC OS 
bash start.sh 
```

Or manually:

```bash
python3 main.py --airports airports.csv --routes routes.csv --output airport_routes_map.html
```


To run visualization with disruptions:

```bash
python app.py
```
